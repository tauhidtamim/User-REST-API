## Language and Tools

- Java 
- Spring Boot
- Postman
- Spring Tool Suite - 4
- MySQL Workbench


## Description

 The REST API can process two types of requests - 

 1) POST /users : store user data in the MySQL database and return the user id as response.

 2) GET /users/{id} : return information about user with user_id = {id}.
